# Mac-React

## How to install backend

Python 3.7
$ pip install pipenv
$ pipenv shell
$ pipenv install django

## How to run backend

$ cd backend
$ pipenv shell
$ python manage.py runserver

## How to install frontend

npm
Yarn

$ cd frontend
$ yarn add bootstrap reactstrap axios react-player react-router-dom

## How to run frontend

$ cd frontend
$ yarn start
